# 連続日数（ストリーク）表示（Issue #11）

習慣を「何日連続で達成したか」を 🔥 N日 で表示し、継続のモチベーションを高めた。

## 何ができるようになったか
- 各習慣の行の右端に現在の連続日数を 🔥 N日 で表示
- 連続が 0 の習慣にはバッジを出さない
- 「今日まだ未完了」でも、昨日までの連続は当日いっぱい維持して表示
  （朝に 0 に見えて萎えるのを防ぐ）

## 設計
- 純粋関数を `src/lib/completion.ts` に追加:
  - `previousDateKey(key)` … 前日のキー。Date 経由で月またぎ・うるう年・年またぎを正しく処理
  - `currentStreak(habit, todayKey)` … 起点（今日 or 昨日）から完了が途切れるまで遡って数える
- `useHabits` が `streakOf(habit)` を公開。UI は日付計算を知らず数字を受け取るだけ。

## 起点の決め方（今日未完了の扱い）
```
今日が完了済み → today を起点に遡る
今日が未完了   → yesterday を起点に遡る（当日いっぱいは連続を維持）
```

## テスト
- `previousDateKey` … 通常 / 月またぎ / うるう年 / 年またぎ
- `currentStreak` … 0件 / 今日のみ / 3連続 / 今日未完了 / 途切れ / 欠け / 月またぎ
- 計 25 件パス（既存含む）。
