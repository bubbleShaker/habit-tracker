# 習慣の削除（Issue #13）

一度追加した習慣を削除できるようにした。

## 何ができるようになったか
- 各習慣の行の右端に ✕ ボタン。タップで一覧から削除し、AsyncStorage からも消える。
- 削除ボタンは独立した Pressable にして、行の「今日やったトグル」と衝突しないようにした。

## 設計
- `useHabits` に `removeHabit(id)` を追加。該当 id を除外するだけで、保存は既存の useEffect が自動で行う。
- UI は明示的な ✕ ボタン（Web でも確実に動くため。長押し/スワイプは Web で不安定なので採用せず）。
- `hitSlop={8}` で小さいボタンでも押しやすく。

## スコープ外（今後の Issue 候補）
- 削除前の確認ダイアログ（Web の Alert は挙動が限定的なため別途検討）
