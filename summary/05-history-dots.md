# 履歴表示（直近7日の達成ドット）（Issue #17）

各習慣の最近の達成傾向を、直近7日分のドットで一目で分かるようにした。

## 何ができるようになったか
- 各習慣の行に直近7日分のドット（●）を古い順（左=6日前 … 右=今日）で表示。
- 達成日は青塗り、未達成日は薄いグレーで表示。

## 設計
- 純粋関数 `lastNDays(todayKey, n)` を `src/lib/completion.ts` に追加。
  - 今日を含む過去 n 日分の `"YYYY-MM-DD"` を古い順で返す（月またぎも `previousDateKey` で正しく処理）。
- `useHabits` に `recentHistory(habit, n=7)` を公開。各日を `isCompletedOn` で判定し boolean[] を返す。
  - UI は date を一切知らず、boolean の並びだけでドットを描ける（関心の分離を維持）。
- UI: メタ行の下にドット行を追加。

## なぜフルカレンダーにしないか
月グリッドは月送り等でスコープが大きい。一覧行内に収まる直近7日ドットで「小さく動くもの」を先に出した。本格カレンダーは別 Issue。

## テスト
- `lastNDays`: 古い順で n 日分 / 末尾は今日 / 月またぎ
- 計 33 件パス。
